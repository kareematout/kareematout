## Selected Work

### Leechy

Production engineering across a rental and services marketplace spanning web,
iOS, payments, authentication, databases, infrastructure, and releases.

- Improved LCP from 3.1s to 2.1s
- Improved p95 API latency from 780ms to 460ms
- Integrated Stripe, PayPal, Apple Pay, Google Pay, and Venmo
- Built secure authentication, database migrations, real-time updates, and
  cross-platform workflows
- Led iOS architecture and release work using SwiftUI, MVVM, and Swift
  Concurrency

### ResuTrace

An end-to-end AI resume platform for job analysis, evidence-grounded
recommendations, structured editing, human-reviewed changes, versioning, and
ATS-friendly PDF generation.

- Built 45+ REST APIs and 100+ React components
- Implemented RAG with OpenAI embeddings and PostgreSQL pgvector
- Built content and layout agents with explicit accept/reject workflows
- Added safeguards for unsupported claims, prompt injection, stale versions,
  and malformed outputs
- Developed 572 automated unit, integration, component, and Playwright E2E tests

[View the ResuTrace case study](https://portfolio-kareem-atout.web.app/projects/ResuTrace)

### CloudReferee

An evaluation-first Infrastructure-as-Code review system that combines
deterministic Terraform and Checkov analysis with evidence-constrained AI
explanations.

- Parses and indexes Terraform/HCL before AI processing
- Normalizes and deterministically groups Checkov findings
- Validates evidence references and finding preservation
- Rejects unsupported AI claims
- Separates original and redacted source representations

[View the CloudReferee case study](https://portfolio-kareem-atout.web.app/projects/CloudReferee)
